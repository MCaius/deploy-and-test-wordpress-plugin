# Security Policy

## Supported Versions

Security updates are provided for the latest released version of Deploy & Test.

| Version | Supported |
| ------- | --------- |
| Latest release | Yes |
| Older releases | No |

## Reporting a Vulnerability

If you discover a security vulnerability in Deploy & Test, please do not open a public issue.

Report it privately using GitHub Security Advisories, or contact the maintainer directly.

Please include:

- A clear description of the vulnerability
- Steps to reproduce the issue
- The affected plugin version
- Any relevant screenshots, logs, or proof of concept
- Whether the issue requires administrator, editor, or unauthenticated access

I will review valid reports as soon as possible and publish a fix before disclosing technical details publicly.

## Scope

This security policy covers the Deploy & Test WordPress plugin code in this repository.

Issues in GitHub Actions workflows, WordPress core, third-party plugins, hosting configuration, or external GitHub repositories are outside the direct scope unless they are caused by this plugin.
